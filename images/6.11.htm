TIP-ON BLUMOTION

Восхитительное сочетание двух технологий

TIP-ON BLUMOTION объединяет преимущества механической системы открывания TIP-ON и надежной системы амортизации BLUMOTION. Для открывания мебели достаточно слегка коснуться фасада. При закрывании c размаху активируется функция TIP-ON. BLUMOTION обеспечивает мягкое и бесшумное закрывание. Главная особенность – это полностью механическое решение.

Обзор применения

TIP-ON BLUMOTION можно использовать с системами выдвижения LEGRABOX, MERIVOBOX и TANDEMBOX, а также с системой скрытых направляющих MOVENTO.

TIP-ON BLUMOTION для LEGRABOX

Элегантную систему выдвижения LEGRABOX можно использовать с TIP-ON BLUMOTION.

Подробнее
TIP-ON BLUMOTION для MERIVOBOX

Система выдвижения MERIVOBOX в сочетании с механической системой открывания предлагает еще больше комфорта.

Подробнее
TIP-ON BLUMOTION для TANDEMBOX

Проверенную временем систему выдвижения TANDEMBOX можно оснастить механической системой открывания.

Подробнее
TIP-ON BLUMOTION для MOVENTO

Систему скрытых направляющих MOVENTO можно использовать с TIP-ON BLUMOTION.

Подробнее

Восхитительное сочетание

Открывание легким нажатием, плавное скольжение, мягкое и бесшумное закрывание. TIP-ON BLUMOTION обеспечивает новый уровень комфорта использования мебели.﻿ Это полностью механическое решение.

Небольшой зазор между корпусом и фасадом

TIP-ON BLUMOTION можно использовать при минимальном зазоре между корпусом и фасадом 2,5 мм, что оптимально подходит для оснащения кухни.

Умная механика

TIP-ON BLUMOTION обеспечивает комфорт использования и свободу выбора: открыть ящик можно также, потянув за фасад, а при закрывании можно прижать его рукой.

Четырехмерная регулировка﻿

Четырехмерная регулировка позволяет создать точный рисунок фасадов. Регулировка по глубине встроена в механизм и осуществляется без инструмента с помощью регулировочного колесика.
